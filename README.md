# Moneta-finance
Aplicação de controle financeiro em Java com testes funcionais e avaliação de dados (projeto acadêmico FIAP).

## Tecnologias
- Java
- Oracle
- SQL

## Atividades realizadas (QA)
- Testes funcionais de cadastro, login e relatórios
- Criação de cenários positivos e negativos
- Registro de bugs
- Validação de dados no banco
